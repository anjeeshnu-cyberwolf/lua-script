Port Scanning Script in Lua

Description: Simple port scanner script written in Lua.

Author: Anjeeshnu

Categories: default, safe

Usage

Usage: nmap -sC -sV -p portrange --script=scriptname target

Example: nmap -sC -sV -p 1-100 --script=portscan.lua localhost

Output

    Nmap scan report for localhost (127.0.0.1)
    Host is up (0.000038s latency).
